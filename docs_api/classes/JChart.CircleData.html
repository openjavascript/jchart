Defined in ../modules/JChart.CircleData/0.1/CircleData.js:163

图表数据显示块的高度 ( pending )
有时需要把多个数据图表显示一样的视觉高度, 但是其他图表的 水平label高度不一样, 就会导致视觉上的不同
这个属性就是为了解决这个问题, 把图表数据显示块设置为统一的高度

Default: 0, 0 = auto
